|                 |                                       |
| --------------- | ------------------------------------- |
| :q              | to `quit` the file                    |
| :e!             | to `trash` all changes and **reload** |
| :q!             | to `trash` all changes and **quit**   |
| :wq             | to `save` the changes and **quit**    |
| ZZ (Shift + zz) | to `save` the changes and **quit**    |
|                 |                                       |
