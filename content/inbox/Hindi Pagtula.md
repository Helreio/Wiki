# Hindi Pagtula

Kung ang tula

Ay kung paano ako magsalita

Sa araw-araw

Bakit pa kailangang pag-aralan ang pagtula?

Hindi pagtula

Kundi pagiging totoo

Pagiging malaya

Ang kailangan kong matutunan

Walang kurikulum

Walang lektyur

Walang workshop

Ang kasintahimik

Ng parang na dinarayo ko

Para marinig ang boses kong

Hindi ko marinig

Kaya di ako makatula

Walang guro

Ang makapagtuturo

Ng kung paano ako

Magiging ako
